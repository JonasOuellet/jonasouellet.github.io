Bio

Hello, my name is Jonathan Ouellet. I am technical animator currently working at Gearbox Software Quebec. I have been working in the game industry since 2013. I started working at Ubisoft Quebec where I contributed on games like Assassin's Creed Freedom Cry and Assassin's Creed Syndicate. I like writing tools to increase production speed and facilitate workflows.

I know Maya and Motion Builder well. I can easily write C++ and python tools for those. I am comfortable with using C++, Python, JavaScript and TypeScript. I know a little bit of mel but I prefer using python by far. I also recently learned how to use ZScript to build tools for ZBrush.

This site is a repository of my personal works.
